=====ВАЖНО!!!=====
1. Папка с именем "Resources" (причем любой вложенности) будет автоматически распознана Unity как специальная папка и весь контент из нее, включая зависимости, будет упакован в конечный билд приложения, даже если этот контент никогда не будет использоваться.
2. Скрипты Controllers, DataBase, Structs - главные скрипты(используются очень часто). Остальные скрипты должны быть в папках соответствующих названию сцен.
==================

1. Assets - все главные файлы для проекта создаются в этой папке
	1.1 Scenes - все сцены проекта
		1) Loading - главная сцена. С нее запускать Unity проект
		2) Authorization - сцены авторизации/ регистрации
		3) Game - сцены игры
	
	1.2 Scripts - все скрипты проекта(название папки = наванию сцены)
		1) Controllers - скрипт для запросов на сервер
		2) DataBase - скрипт для работы с БД
		3) Structs - набор структур которые отправляются на сервер и принимаются от сервера
		4) Authorization - папка скриптов авторизации, сцен Login и Register
		5) Loading - папка скриптов сцены Loading
		6) UserOnline - скрипт который нужен, чтобы в 1 аккаунт не смогли зайти 2 человека(вешать на все менеджеры сцены где нужна проверка)
		7) Game - папка скриптов игрового процесса
			7.1) BaseObjects - папка где находятся базовые классы сущностей
			7.2) Player - папка скриптов игрока
			7.3) Inventory - папка скриптов инвенторя
			7.4) Items - папка скриптов предметов в игре
			7.5) Camera - папка скриптов управления камерой
			7.6) Attack - папка скриптов атак
			7.7) Effects - папка скриптов эффектов
	
	1.3 InternalAssets - все файлы созданные нами
		1) Animations - анимации (скорее всего нужно будет разделить папку на: графы и исходники анимации)
			1.1) Player - анимации игрока
		2) Audio - звуки
		3) Fonts - шрифты
		4) Images - 2D-изображения
			4.1) UI - изображения интерфейса
				4.1.1) Atlases - набор атласов UI
				4.1.2) Sources - Исходные изображения для атласов UI. Каждая внутренняя папка должна иметь тоже имя, что и атлас
		5) Materials - материалы игры
		6) Prefabs - префабы
		7) Resources - ресурсы (Они загружаются динамически, чем меньше папка - тем лучше)
		8) Sprites - спрайты 
		9) ScriptableObject - папка для хранения/сохранения больших объемов данных
	
	1.4 ExternalAssets - все файла скаченные из вне(сторонние ассеты)
		1) Photon - ассет для мультиплеера
	
	1.5 Plugins - папка плагинов
		1) плагин БД
