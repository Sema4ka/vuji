\doxysubsection{Подробное описание}
Модуль для всего, что связанно со считыванием нажатий на клавиатуру (привязка, считывание, регулирование) 

All\+Keys -\/ Список всех Key\+Code 

movement\+Keys -\/ Список всех действий связанных с движением 

ability\+Keys -\/ Список всех действий связанных с умениями персонажа 

ui\+Keys -\/ Список всех действий связанных с меню UI 

keybinds -\/ Все связанные действия и Key\+Code 

numbers\+Key\+Codes -\/ Key\+Code для цифровых клавиш клавиатуры
